Cloud Print Using Pi

Low cost cloud print server ever!

It is possible to get some benefits of cloud printing without buying a new printer and having a dedicated computer to serve as print server. Here, we are going to try to set up cloud print server with Raspberry pi. Hence we can enjoy cloud print on the go by following simple few steps.
